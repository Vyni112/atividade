Introduçao HTML

  • HTML ou Hipertext Markup Language.

  • E um codigo ultilizado para descrever a estrutura de um documento, msd nao a verdadeira aporesentaçao dele.

  • O HTML diz ao navegador tudo sobre o conteudo recebifo e s estrutura da pagina.

  • Ele definie o cabeçalho, paragrafos, listas, tabelas entre muitos outros elementos, por meio das TAGS.

Tags HTML

  • O HTML tem suas proprias Tags onde nela podemos definir uma certa propriedades para os campos, dentre elas estao:

      • < html> Define um documentom HTML.

      < body> Define um corpo para o documento.

      < h1> a <h6> Define titulo de 1 a 6.

      < p> Define um paragrafo.

      < br> Insere uma unica quebra de linha.

      < hr> Define um filete de horizontal.

Estruta de uma pagina HTML

  • Toda pagina html, por mais simples que seja, deve conter as seguintes tags elementares:

      • < body> e a tag que apresenta informaçoes gerais sobre a pagina e de configuraçao

      < tilte> e a tag que informa o titulo de uma pagina na barra do navegador;

      < body> E a tag que delimita o conteudo da pagina.
